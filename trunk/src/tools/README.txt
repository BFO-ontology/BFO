In this directory

bfo.asd - asdf specification to load the lisp files necessary

bfo2-reference.lisp - specification of structure of bfo2, URIs, class
   hierarchy, property hierarchy, mapping from 1.1 etc

raw-schulz-axioms.lisp - The non-annotation axioms extracted from owl-schulz/bfo.owl
filter-schulz-axioms.lisp - Code to sort them into groups 
   a) to review 
   b) generated by bfo2-reference (not needed) 
   c) axioms that use a term not in the spec (yet?)
filtered-schulz-axioms.lisp - the result of running the above code

extract-reference-annotations.lisp - code to extract annotations from
   ms word bfo2 reference

raw-extracted-from-spec.txt - dump of annotations, for inspection

** From an earlier effort resulting in owl-ruttenberg
bfo-relations.lisp
bfo1.1-to-2.lisp

temporalized-relations.lisp - prototype of temporalized relations and
   temporally qualified continuants.
