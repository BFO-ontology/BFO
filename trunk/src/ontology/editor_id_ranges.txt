
BFO NUMBER CHECKOUT LIST

File to contain BFO ID number reservations.  Edit the file and state what
numbers you are taking.  This way other curators at your site will not
use the same numbers.

BFO_0000100	BFO_0000199	LBL	cjm	Relation IDs
BFO_0010000	BFO_0019999	BFO	BFO	Axiom IDs
